# 🌦 Weather Data Analysis Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive **Power BI dashboard** built to analyze weather data and uncover patterns in key weather indicators such as temperature, humidity, wind speed, and precipitation.

The dashboard enables users to explore weather trends through interactive visualizations and filters, making it easier to identify patterns and insights from the dataset.

---

## 🎯 Objectives
- Analyze weather trends over time
- Compare different weather parameters
- Build an interactive dashboard for easy data exploration
- Practice data cleaning, transformation, and visualization using Power BI

---

## 🛠 Tools & Technologies
- **Power BI**
- **Power Query**
- **DAX**
- Data Visualization
- Data Cleaning

---

## 📊 Dashboard Features
- Interactive filters for exploring weather conditions
- Visual comparison of temperature, humidity, and wind speed
- Clear charts and KPIs to highlight important metrics
- Easy-to-understand layout for quick insights

---

## 📈 Key Insights
- Temperature patterns across different time periods
- Relationship between humidity and precipitation
- Wind speed variations
- Seasonal weather trends
